# mobnews

to run the project

```
npm i
npm start
```

it will run at `localhost:3000`

# project structure

```
src
|
|_ assets
|
|_ pages
|
|_ components 

```

asset เอาไว้ใส่รูป ใส่ฟ้อนท์

pages เอาไว้ใส่หน้าใหม่ๆ (ตอนนี้มีหน้าเดียว)

components เอาไว้ใส่ component, container, wrapper to be reused
